# Code of Conduct

Git Diff Lens welcomes practical, respectful collaboration.

## Our Standards

Expected behavior:

- Be respectful and direct.
- Assume good intent, then ask clarifying questions.
- Keep reviews focused on code, evidence, user impact, and maintainability.
- Be careful with security-sensitive details and private repository information.

Unacceptable behavior:

- Harassment, personal attacks, or discriminatory language.
- Publishing another person's private information or credentials.
- Posting exploit details or sensitive logs in public issues.
- Repeatedly derailing technical discussions.

## Enforcement

Maintainers may edit, hide, or remove comments, issues, pull requests, or other contributions that violate these standards. Serious or repeated violations may result in being blocked from project spaces.

If you need to report a conduct issue, contact a maintainer through the repository's GitHub channels.
